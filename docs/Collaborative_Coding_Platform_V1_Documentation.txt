
Collaborative Coding Platform - V1 Documentation

Project Goal
A production-inspired collaborative coding platform built with free technologies.

Tech Stack
Frontend: React, Vite, Tailwind CSS, Monaco Editor, Socket.IO Client
Backend: Node.js, Express, Socket.IO, MongoDB Atlas, Mongoose

V1 Features
- Room System
- Real-Time Code Sync
- User Presence
- MongoDB Persistence
- JavaScript Execution
- Python Execution
- C++ Execution
- Collaborative Output
- Language Selector
- Output Persistence
- Language Persistence
- Unique Execution Files

Database Schema (Room)
- roomId
- code
- language
- lastOutput
- createdAt
- updatedAt

Architecture
Frontend
 -> Socket.IO + REST API
 -> Express Server
 -> MongoDB + Execution Service

Execution Flow
Run Button
 -> API
 -> Controller
 -> Execution Service
 -> Runtime
 -> Output
 -> Broadcast to Room
